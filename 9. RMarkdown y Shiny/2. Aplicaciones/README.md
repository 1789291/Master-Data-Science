# Índice de aplicaciones Shiny

1. **Aplicacion_MFZ**: histograma con cambio del número de bins.

2. **Aplicacion_size_puntos**: scatter plot con cambio del tamaño de los puntos.

3. **interfaz**: interfaz más avanzada (varios inputs, etc...).

4. **interfaz2**: creación de tabs.

5. **inputs**: ejemplos de botones para inputs.

6. **sencillo**: ejemplo de regresión con scatter e histograma.

7. **guardar**: guardar elementos de la aplicación.

8. **reactividad**: ejemplo con reactividad.

9. **Ejercicio_GDP**: ejercicio de práctica.
